AniHub CLI

Неофіційний термінальний клієнт для anihub.in.ua — пошук, бібліотека й перегляд у mpv. Написаний на Rust 🦀

GitHub Releases README (UK)

Можливості

  • Пошук AniHub (strict / extended) + франшизи
  • Бібліотека зі статусами, фільтрами, resume
  • mpv (Ashdi), MoonAnime у браузері
  • Discord Rich Presence (opt-in)
  • Теми RGB / ANSI, кеш постерів і метаданих
  • Шорткати на EN і UA/RU розкладках

Встановлення (Linux / macOS)

curl --fail --location --retry 3 \
  https://raw.githubusercontent.com/NEO-LAX/anihub-cli/main/install.sh | bash
